# Welcome to Blumbaben

Blumbaben is a lightweight TypeScript React hook library for adding contextual formatting toolbars to text inputs and textareas. Show contextual formatting options when users focus on input fields.

## Core concepts

### Global state pattern

Blumbaben uses a singleton pattern to manage toolbar state globally. This ensures only one toolbar is visible at a time across your entire application, providing a clean and focused user experience.

```tsx
// Multiple inputs automatically share the same toolbar
<input {...getInputProps()} />
<textarea {...getInputProps()} />
<input {...getInputProps()} />

{/* Single toolbar renders conditionally */}
{isVisible && <div>Toolbar content</div>}
```

### Formatter functions

The core pattern of Blumbaben is the `FormatterFunction` - a simple function that transforms text:

```typescript
type FormatterFunction = (text: string) => string;

// Examples
const bold = (text: string) => `**${text}**`;
const uppercase = (text: string) => text.toUpperCase();
const trim = (text: string) => text.trim();
```

### Main exports

Blumbaben provides multiple ways to integrate formatting toolbars into your application:

- **`useFormattingToolbar`** - Main hook with full functionality
- **`useFormattingToolbarState`** - Lightweight state-only hook for toolbar components
- **`FormattingToolbar`** - Ready-to-use component with automatic positioning
- **`withFormattingToolbar`** - Higher-order component for wrapping existing inputs
